## CSS Validation

The CSS stylesheet for the FreshCharge+ website was tested using the **W3C CSS Validator** (CSS Level 3 + SVG).  
The file passed validation with **no errors**, ensuring clean and compliant CSS code.

Two warnings were shown, but neither affects website performance or accessibility:

- Imported style sheets are not checked in direct input upload mode  
- The `clip` property is deprecated  

These warnings do **not** impact the visual design or functionality of the site.

## HTML Validation {#html-validation}

All HTML pages of the FreshCharge+ website were validated using the  
**W3C Markup Validation Service**:  
https://validator.w3.org/nu/#textarea

Each page was checked through the *“Validate by Direct Input”* option, and **all pages returned with no HTML errors**.
